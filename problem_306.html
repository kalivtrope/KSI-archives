Zadání

Jupyter Notebook (https://jupyter.org/), dříve známý jako iPython Notebook, je interaktivní webové rozhraní pro práci (nejen) s Pythonem a jeho knihovnami třeba pro zpracování dat, nebo kreslení grafů.

Kromě Jupyter Notebooku ještě existuje nadstavba JupyterLab, kterou v úloze používat nebudeme.

Laboratorní deník

Uživatelské rozhraní Jupyter Notebooku je inspirované laboratorním deníkem, kde se vzájemně střídají textové popisy, obrázky a výpočty, podle toho, jak je zrovna potřeba. První s takovým rozhraním přišel v roce 1988 program Wolfram Mathematica, který navrhl britský vědec a tak trochu egomaniak Stephen Wolfram. Anglický název pro toto uživatelské rozhraní je “notebook interface”, česky můžeme tomuto typu dokumentu říkat „výpočetní blok“.

Blok v Jupyteru sestává z buněk dvou typů, textových (Markdown) a programových (Code).

Textová buňka dovoluje vepisovat text formátovaný pomocí značkovacího jazyka Markdown.

Programová buňka dovoluje vepisovat zdrojový kód Pythonu a hned ho i spouštět. Výstup z programu (výpisy pomocí funkce print() a také výsledek vyhodnocení posledního příkazu v buňce) se zobrazuje pod příslušnou buňkou. Kromě textu mohou programové buňky generovat grafy, obrázky, nebo i zvuk. Tyto výstupy se také zobrazují bezprostředně pod programovou buňkou, která je vygenerovala.

Pokud definuješ funkci nebo proměnnou v libovolné programové buňce, a buňku spustíš, bude funkce nebo proměnná následně dostupná ze všech ostatních programových buněk v bloku.

Instalace Jupyter Notebooku

Instalační instrukce najdeš přímo v nápovědě k Jupyter Notebooku, nebo také zde v úloze.

Nejjednodušší je nainstalovat si Pythonovou distribuci nazvanou Anaconda, (vydává ji stejnojmenná firma, ale její použití je možné zdarma), která Jupyter Notebook obsahuje už v základu. Pokud se rozhodneš nainstalovat Anacondu, seznam se s dokumentací k Anaconda Navigatoru, což je grafický spouštěč aplikací začleněných do distribuce Anaconda. Kromě Jupyter Notebooku tam najdeš třeba VSCode, což je velmi dobré integrované vývojové prostředí pro Python i další programovací jazyky od firmy Microsoft.

Protože ty už na svém počítači Python máš, a protože už víš jak pracovat s pipem, může ti přijít jednodušší instalace pomocí pipu. Asi si budeš chtít pro Jupyter vytvořit virtuální prostředí. Pro začátek to silně doporučuji.

python3 -m venv muj_jupyter3_venv
# na Linuxu načti skript activate
source muj_jupyter_venv/bin/activate
# na Windows spusť activate.bat, případně activate.ps1
muj_jupyter_venv/Scripts/activate.bat

Potom už můžeš nainstalovat Jupyter. Instalace bude chvilku trvat.

pip3 install --upgrade pip
pip3 install jupyter

Tvůj první blok

Vytvoř nový blok

Na webové stránce si najdi složku, kde chceš vytvořit nový soubor s blokem, a klikni na New > Notebook > Python 3 vpravo nahoře.

Otevře se nový dokument s jednou, programovou (typu Code), buňkou.

Vyhodnoť programovou buňku

Napiš do ní 3 + 4 a stiskni klávesovou zkratku Ctrl + Enter

Přidej textovou buňku

Klikni na Insert > Insert Cell Above. Tím vložíš novou buňku na začátek dokumentu.

Označ nově vloženou buňku, a klikni na Cell > Cell Type > Markdown. Tím ji změníš na textovou buňku (typu Markdown).

Vepiš do nové buňky následující. Klávesa Enter přesune kurzor na nový řádek, Ctrl + Enter ukončí editaci. Pokud chceš text dále upravovat, stačí na buňku dvojkliknout.

# Jirkův blok

kde se počítá, kolik je $3 + 4$

Až ukončíš editaci buňky (pomocí Ctrl + Enter), vykreslí se buňka tahle.

Formátování textu a matematiky

Jak formátovat text se dozvíš v nějakém představení formátu Markdown, například tenhle je docela pěkný a v češtině.

Jak formátovat matematiku, mezi znaky $, se dozvíš v nějaké příručce k programu LaTeX (čti latech, popřípadě anglicky lejtek, protože to X na konci je řecké písmeno chí). Například návod od Roberta Maříka v části „Základní výrazy“ obsahuje hodně ukázek. Knihovna, kterou Jupyter používá pro vykreslování matematiky na webu, se jmenuje MathJax. Pod tímto heslem také najdeš užitečnou nápovědu, zvláště k pokročilým funkcím.

Jupyter Notebook v Internetu

Když novinář Jan Cibulka chtěl v roce 2014 v Jupyter Notebooku (tehdy ještě iPython Notebooku) zpracovat data o veterinárních inspekcích v masnách, nainstaloval si vše potřebné na virtuální stroj ve službě Microsoft Azure. Na svém počítači měl spuštěný jen webový prohlížeč, ze kterého se k Jupyter Notebooku připojil. Tehdy nakonec firmě Microsoft za virtuální stroj zaplatil celkem 15 korun.

V dnešní době si můžeš spustit vzdálený Jupyter Notebook mnohem jednodušeji, už předinstalovaný, a dokonce zcela zdarma.

Když jsem hledal, které služby představit, našel jsem jich docela dost, namátkou Binder, Code Ocean, Google Colaboratory, Gigantum, Nextjournal nebo JetBrains Datalore.

Colaboratory i Datalore jsou zdarma, navíc Colaboratory má pěknou integraci s Google Drive, a Datalore je dílo sice ruské firmy, ale sídlící v Praze. Takže pokud se rozhodneš spouštět Jupyter Notebook vzdáleně, doporučuji využít jednu z těch dvou.

Úvod do Google Colaboratory (Datalore)

Otevři si Colaboratory nebo Datalore, vyplň registraci, a založ nový Notebook (v terminologii Datalore je to Workbook).

Do první programové buňky vlož následující, a stiskni Ctrl + Enter.

import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 2, 100)

plt.plot(x, x, label='linear')
plt.plot(x, x**2, label='quadratic')
plt.plot(x, x**3, label='cubic')

plt.xlabel('x label')
plt.ylabel('y label')

plt.title("Simple Plot")

plt.legend()

plt.show()

Výsledkem by měl být graf jako na následujícím obrázku.

Pokud to stejné zkusíš v Jupyter Notebooku na svém počítači, pravděpodobně dostaneš chybovou hlášku. To je proto, že nemáš nainstalovanou knihovnu matplotlib pro kreslení grafů, ani numpy, pro práci s poli a maticemi čísel. Pokud používáš Anacondu, tak chybu neuvídíš, Anacona má totiž spoustu věcí už v základu předinstalovaných.

Jak v Jupyter Notebooku instalovat balíčky

V Colaboratory a Jupyter Notebooku (na vlastním počítači) do programové buňky napiš následující, a spusť ji.

import sys
!{sys.executable} -m pip install matplotlib numpy

Datalore nerozumí Jupyterové konvenci s vykřičníkem, a proto se tam na to musí trochu jinak, takhle.

import subprocess
import sys
print(subprocess.getoutput(f"{sys.executable} -m pip install matplotlib numpy"))

Proč tak složitě, proč ne jen !pip install matplotlib numpy? Protože takto je jistota, že se použije pip ve stejném virtuálním prostředí, v jakém běží Jupyter Notebook.

Jak do Jupyter Notebooku nahrát soubory?

Datalore má v menu akci pro upload souborů. Pokud z nějakého důvodu chceš spouštět systémové příkazy, dá se to následujicím způsobem.

import subprocess
print(subprocess.getoutput("wget https://nlp.fi.muni.cz/trac/research/raw-attachment/wiki/en/AdvancedNlpCourse2015/reviews_czaccent.csv"))

Co se Colaboratory týče, jejich hostovaný Jupyter Notebook pravděpodobně poběží v nějaké verzi Ubuntu Linuxu. Systémové příkazy je možné spouštět tak, že je vepíšeš do programové buňky, a uvedeš před ně znak vykřičníku.

!wget https://nlp.fi.muni.cz/trac/research/raw-attachment/wiki/en/AdvancedNlpCourse2015/reviews_czaccent.csv

Soubor reviews_czaccent.csv se uloží do aktuálního adresáře. Nyní už můžeš data načíst, a dále zpracovávat.

import pandas as pd
data = pd.read_csv("reviews_czaccent.csv", header=None, error_bad_lines=False)
data.head()

Zarazilo tě to error_bad_lines=False? Mělo by, protože takto vlastně nevíme, co za data načítáme. Také z výpisu prvních 10 řádků v Pandasovém DataFrame je vidět, že jsou nějaká divná. Před tím, než můžeš s těmito daty něco dál dělat, musíš přijít na to, jak je pomocí funkce pd.read_csv načíst správně, jak odfiltrovat neplatné řádky, a zda odfiltrování neplatných řádků vlastně data neznehodnotí.
