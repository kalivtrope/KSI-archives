Zadání

Sob se vrátil z Brna a začal přemýšlet nad letošním InterSoBem. Zamýšlel se hlavně nad tím, kolika způsoby mohla být obsazená první tři místa. Zjistil si, že to je docela jednoduché.

V letošním ročníku se zúčastnilo 32 týmů. Na prvním místě mohlo být 32 týmů. Když je vybraný ten na prvním místě, zbývá 31 týmů, které mohou být na místě druhém. A nakonec 30 týmů, které mohou obsadit místo třetí. Dohromady to dává 32⋅31⋅3032⋅31⋅30 možností.

Pak si Sob ale řekl, že by se hodilo mít způsob, jak něco takového počítat obecně. Předchozí úvahu by mohl zobecnit na to, že když potřebuje spočítat, jaký je počet uspořádaných kk-tic z nn týmů (tedy kolika způsoby mohly týmy obsadit prvních kk pozic), bude postupně vybírat první, druhý, až kk-tý tým. Tomu odpovídá vzoreček

n⋅(n−1)⋅(n−2)⋅…⋅(n−k+1),n⋅(n−1)⋅(n−2)⋅…⋅(n−k+1),
protože pro každou další pozici máme o 1 tým méně, ze kterých můžeme vybírat. Díky vykrácení se tento vzoreček dá napsat jako
n⋅(n−1)⋅(n−2)⋅…⋅(n−k+1)⋅(n−k)⋅(n−k−1)⋅…⋅2⋅1(n−k)⋅(n−k−1)⋅…⋅2⋅1,n⋅(n−1)⋅(n−2)⋅…⋅(n−k+1)⋅(n−k)⋅(n−k−1)⋅…⋅2⋅1(n−k)⋅(n−k−1)⋅…⋅2⋅1,
 což se dá napsat už v kratším tvaru jako
n!(n−k)!.n!(n−k)!.

Abychom měli představu, co jsou to variace bez opakování, vypíšeme si prvně všechny variace bez opakování třetí třídy ze 4 prvků (zneužijeme množinu {a,b,c,d}{a,b,c,d}). Množina obsahující tyto variace vypadá následovně {(a,b,c),(a,b,d),(a,c,b),(a,c,d),(a,d,b),(a,d,c),{(a,b,c),(a,b,d),(a,c,b),(a,c,d),(a,d,b),(a,d,c), (b,a,c),(b,a,d),(b,c,a),(b,c,d),(b,d,a),(b,d,c),(b,a,c),(b,a,d),(b,c,a),(b,c,d),(b,d,a),(b,d,c), (c,a,b),(c,a,d),(c,b,a),(c,b,d),(d,a,b),(d,a,c),(c,a,b),(c,a,d),(c,b,a),(c,b,d),(d,a,b),(d,a,c), (d,a,b),(d,a,c),(d,b,a),(d,b,c),(d,c,a),(d,c,b)}(d,a,b),(d,a,c),(d,b,a),(d,b,c),(d,c,a),(d,c,b)}. Počet takových variací (počet prvků této množiny) je 4!(4−3)!=4!=4⋅3⋅2=24.4!(4−3)!=4!=4⋅3⋅2=24.

Definice:
Variace bez opakování k-té třídy z n prvků je uspořádaná k-tice z n prvků taková, že se v ní žádný prvek neopakuje.

Počet variací bez opakování k-té třídy z n prvků je roven n!/(n-k)! (kde ! značí faktoriál).

Definice variace bez opakování a variace s opakováním se mohou zdát podobné, ale jak už to bývá, používají se většinou na docela odlišné věci. V obou případech se jedná o výběr uspořádaných kk prvků z nn-prvkové množiny. Rozdíl je ovšem to, že ve variacích s opakováním se prvky mohou opakovat (dva kamarádi mohou mít stejnou příchuť zmrzliny), kdežto ve variacích bez opakování se prvky opakovat nemohou (tým nemůže být na prvním i druhém místě, pokud nebereme v potaz remízy).

Sob si chce ověřit, jestli jeho předchozí výpočet sedí se vzorečkem, který si odvodil. Stejně jako předtím bude vybírat první tři místa ze 32 týmů. Tedy k=3k=3 a n=32n=32. Takže teď Sobovi stačí už jenom dosadit do n!(n−k)!n!(n−k)!, tedy 32!(32−3)!=32!29!=32⋅31⋅30=2976032!(32−3)!=32!29!=32⋅31⋅30=29760.

Tato úloha již byla uzavřena a není možné ji odevzdat. Navazující úlohy ti však byly automaticky odemčeny. Navíc se můžeš podívat na vzorové řešení úlohy.

Řešení

Tady je jedno z možných řešení, které vychází z rekurzivního řešení variation_with_repetition z předešlé úlohy.

def variation_without_repetition(k, n):
    """Returns all variations without repetition of k-th class
    from numbers from 0 to n-1.
    """
    if k == 0:
        return [[]]
    if k < 0 or n < k:
        return []

    result = []

    # Get all variation of the (k-1)-th class
    previous_variations = variation_without_repetition(k - 1, n)
    # Add each possible number to each of these variations
    for variation in previous_variations:
        for i in range(n):
            if i not in variation:
                extended_variation = variation[:]
                extended_variation.append(i)
                result.append(extended_variation)
    return result
