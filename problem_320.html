Zadání

V semináři nebudeš úlohy řešit pouze teoreticky, ale i prakticky. Budeš k tomu využívat programovací jazyk Python. Pokud používáš Linux, máš již nejspíš Python na svém počítači nainstalovaný; pokud používáš Windows, stáhni si Python 3. Pokud bys chtěl přesně stejnou verzi jako používáme my, stáhni si Python 3.5.4. Aktuální verze Pythonu 3 (3.8.5), na kterou tě odkazujeme výše, má naprostou většinu věcí stejnou, ale z vylepšení má například lepší chybové hlášky, a prvky ve slovníku se (ve verzi 3.6 a novějších) vždy projdou (třeba cyklem for) v pořadí, v jakém byly vloženy, zatímco ve verzi 3.5 je pořadí průchodu slovníkem náhodné.

Projděme si teď společně základy Pythonu, které budeš potřebovat k řešení úloh.

Práce s čísly

print(2 + 3)               # sčítání
print(2 * 5)               # násobení
print(10 / 3)              # desetinné dělení
print(10 // 3)             # celočíselné dělení
print(10 % 3)              # zbytek po dělení
print((5 - 3) * (7 - 4))   # uzávorkování

Proměnné

x = 2             # do proměnné x přiřaď číslo 2
y = 5             # do proměnné y přiřaď číslo 5
product = x * y   # spočítej součin hodnot proměnných x a y, výsledek přiřaď do proměnné product
print(product)    # vypiš hodnotu proměnné product

Vstup a výstup

# načtení textoveho vstupu od uživatele
name = input('Zadej své jméno: ')
# načtení čísla od uživatele
age = int(input('Zadej svůj věk: '))
# výpis informací
print('Jmenuješ se', name, 'a je ti', age, 'let.')

Podmínky

age = int(input('Zadej svůj vek: '))
if age >= 15:
    print('plné vstupné')
else:
    print('poloviční vstupné')

Opakování

Pokud víme, kolikrát má opakování proběhnout, použijeme for cyklus:

# 5krát vypiš řetězec 'Ahoj!'
for i in range(5):
    print('Ahoj!')

Pokud nevíme, kolikrát má opakování proběhnout, použijeme while cyklus:

# výpis všech mocnin trojky menších než 1000
power = 1            # prvni mocnina je 1
while power < 1000:  # pokračujeme v cyklu, pokud je mocnina stále menší než 1000
    print(power, end=' ')
    power = power * 3  # vypočítáme další mocninu v pořadí

Odsazení

Python je velmi citlivý na odsazování. Zanořené bloky kódu (například těla podmínek či cyklů) musí být odsazeny 4 mezerami. Když nejsou, Python v některých případech zařve IndentationError. To jsou ty lepší případy. Pokud ti bude odsazení chybět jen někde, nebo se ti občas vloudí tabulátor na místo čtyř mezer, Python to bude tiše interpretovat jinak, než by člověk čekal. Pozor tedy na to.

Komentáře

Komentáře jsou důležité! Pomáhají s orientací a zaznamenávají, co která část kódu dělá, což se hodí jak programátorovi, tak těm, kteří kód čtou po něm.

Jednořádkové komentáře v Pythonu začínají znakem #.

# komentáře by měly být stručné a výstižné

Víceřádkové řetězce se používají pro dokumentaci.

"""Podrobnější popis

který vysvětluje, co dělá nějaká funkce, třída či metoda,
vypadá takto.
"""

Knihovny

Občas nám nestačí vestavěné funkce a musíme použít některou z rozšiřujících knihoven. Typickým příkladem je knihovna math, která obsahuje spoustu užitečných matematických funkcí. Funkce, které potřebujeme, importujeme příkazem from jmeno_knihovny import jmeno_funkce.

# z knihovny math importujeme funkci sqrt (odmocnina) a konstantu pi
from math import sqrt, pi
print(sqrt(4))
print(pi)

Můžeme importovat i celou knihovnu příkazem import jmeno_knihovny. Pak je třeba odkazovat se na konkrétní funkce jmeno_knihovny.jmeno_funkce.

Mocniny

Některé jednoduché úlohy budou čistě programovací a ty náš systém umí opravit automaticky. Vyzkoušej si to na následujícím zadání.

Načti od uživatele číslo n a vypiš nejvyšší mocninu čísla n, která je menší než 500. Například pro vstup 2 program vypíše 256 a pro vstup 5 vypíše 125. Můžeš předpokládat, že 2≤n<5002≤n<500. Načtení čísla ze vstupu proveď pomocí n = int(input()).

Co jsme ti ještě nestihli prozradit je fakt, že Python má více různých typů hodnot reprezentujících čísla. Typ int (z anglického integer) je celočíselný s neomezeným rozsahem, zatímco typ float (z anglického floating point, číslo v pohyblivé řádové čárce) má omezený rozsah i přesnost, ale umí reprezentovat desetinná čísla.

>>> int(10.0)
10
>>> float(10.0)
10.0

V této úloze ti budeme dávat výhradně celočíselné vstupy a očekáváme celočíselné výstupy bez desetinné čárky.

Pokud chceš v řešení používat operaci dělení, použij operátor celočíselného dělení, //.

>>> x = 6 / 3
>>> y = 6 // 3

>>> print(x)
2.0
>>> print(y)
2

>>> type(x)
<class 'float'>
>>> type(y)
<class 'int'>
1
2
3
