Zadání

Události, aneb jak poslouchat uživatele

V minulé úloze jsme si řekli, jak něco vykreslit. Neboli jak něco sdělit uživateli. Nyní si povíme, jak reagovat na různé vstupy od uživatele. Funkce pygame.event.get() vrací i jiné události nežli požadavek na ukončení aplikace. Níže si ukážeme základní typy událostí. Kompletní tabulku událostí naleznete například v dokumentaci pygame.

Event type Attributes
QUIT none
KEYDOWN key, mod, unicode, scancode
KEYUP key, mod
MOUSEMOTION pos, rel, buttons
MOUSEBUTTONUP pos, button
MOUSEBUTTONDOWN pos, button

Při zpracovávání události MOUSEBUTTONUP nebo MOUSEBUTTONDOWN se vám bude hodit metoda pygame.mouse.get_pos(), která vám pošle souřadnice kurzoru. Při zpracovávání události KEYDOWN a KEYUP se pro zjištějí, zda konkrétní klávesa byla zmáčknuta se použije hodnota event.key. A pak se porovná s hodnotou příslušné klávesy.

Příklad obsloužení KEYDOWN a KEYUP události.

for event in pygame.event.get():
    if event.type == pygame.KEYUP:
        if event.key == pygame.K_a:
            print('You released key a')

    if event.type == pygame.KEYDOWN:
        if event.key == pygame.K_a:
            print('You pressed key a')

Konstanty kláves jsou ve tvaru K_klávesa. Seznam všech kláves, které pygame podporuje, naleznete v dokumentaci, ovšem názvy jsou intuitivní.

Struktura kódu

Ve hrách se řeší více věcí najednou jako jsou grafika, vstup od uživatele (events), logika hry (validace tahů případně strategie). Proto je vhodné tyto části od sebe oddělit. Pokud hra má více částí (menu, nastavení, samothá hra), pak je dobré tyto části od sebe oddělit také.

Jeden z postupů je pro každou část hry si vytvořit objekt. Potom v každém objektu si vytvořit metody pro vykreslování, upravování herního stavu, zpracovávání událostí a dalších věcí.

Poznamenejme však, že kvalita kódu je subjektivní dojem a v různých hrách je vhodné provádět různou strukturu kódu. (Například tuto strukturu kódu bych doporučil pro hru, kde je mnoho jednodušších levelů. Avšak pokud logika hry bude složitější a hra bude mít jen jednu obrazovku, například budete chtít naprogramovat single player versi se složitou strategií, pak se vyplatí spíš rozdělovat grafickou a logickou část.)

Jinými slovy není důležité, jak přesně má kód vypadat, ale je důležité, aby kód byl strukturovaný a dobře čitelný. Pomůže, pokud se před psaním kódu (nejen v pygame) zamyslíte, jaké věci potřebujete napsat.

Níže můžete vidět příklad, jak by takový kód mohl být strukturovaný. Jak implementovat některé metody si povíme níže.

    screen = pygame.display.set_mode((CELL_SIZE*(size + 2),
                                      CELL_SIZE*(size + 2)))

    done = False
    game_screen = Menu(screen)
    while not done:
        result = game_screen.game_tick()
        if result == "self":

        if result == "quit":

Mohli bychom mít dva objekty, jeden symbolizující menu, druhý samotnou hru. Jak menu, tak hra má svoji kreslící metodu a metodu zpracovávající události.

Jak vytvořit menu?

Nejdříve si povězme, jak vytvořit tlačítko. Z grafické části si tlačítko můžeme představit jako obdélník s textem. Informaci, zda uživatel kliknul na tlačítko zjistíme pomocí událostí. Pokud dostaneme událost typu MOUSEBUTTONUP tak se podíváme pomocí pygame.mouse.get_pos(), zda jsme klikli dovnitř tlačítka. Pokud ano, tak provedeme příslušnou akci. Uvedeme si to na příkladu.

class Menu:
    def __init__(self, screen):
        self.screen = screen
        width, height = pygame.display.get_surface().get_size()
        self.width = width
        self.height = height
        self.button_width = 100
        self.button_height = 20
        self.start_btn_color = (0, 255, 0)
        self.quit_btn_color = (255, 0, 0)
        self.start_btn_pos_x = width // 3 - self.button_width // 2
        self.start_btn_pos_y = height // 2 - self.button_height // 2
        self.quit_btn_pos_x = (2 * width) // 3 - self.button_width // 2
        self.quit_btn_pos_y = height // 2 - self.button_height // 2

    def draw_text(self, text, x, y):
        font = pygame.font.Font(None, 25)
        text = font.render(text, True, BRIGHT_COLOR)
        text_rect = text.get_rect(center=(x, y))
        self.screen.blit(text, text_rect)

    def draw_buttons(self):
        self.screen.fill(BLACK)
        pygame.draw.rect(self.screen, self.start_btn_color,
                         (self.start_btn_pos_x, self.start_btn_pos_y,
                          self.button_width, self.button_height))
        x = self.start_btn_pos_x + self.button_width // 2
        y = self.start_btn_pos_y + self.button_height // 2
        self.draw_text('Start', x, y)
        pygame.draw.rect(self.screen, self.quit_btn_color,
                         (self.quit_btn_pos_x, self.quit_btn_pos_y,
                          self.button_width, self.button_height))
        x = self.quit_btn_pos_x + self.button_width // 2
        self.draw_text('Quit', x, y)

    def check_events(self):
        result = "self"
        for event in pygame.event.get():
            if event.type == pygame.MOUSEBUTTONUP:
                x, y = pygame.mouse.get_pos()
                if x > self.start_btn_pos_x and x < self.start_btn_pos_x +\
                   self.button_width and y > self.start_btn_pos_y and \
                   y < self.start_btn_pos_y + self.button_height:
                    result = "game"
                if x > self.quit_btn_pos_x and x < self.quit_btn_pos_x +\
                   self.button_width and y > self.quit_btn_pos_y and \
                   y < self.quit_btn_pos_y + self.button_height:
                    result = "quit"
            if event.type == pygame.QUIT:
                result = "quit"
                return result
        return result

    def game_tick(self):
        result = self.check_events()
        self.draw_buttons()
        pygame.display.update()
        return result

V metodě __init__ definujeme kde tlačítka budou a jak budou vypadat (definujeme konstanty). V metodě draw_buttons vykreslujeme tlačítka a text na nich. Nakonec v metodě check_events zjišťujeme, jestli se kliklo na nějaké z tlačítek a podle toho vracíme událost. Metoda game_tick slouží ke komunikaci s game loop. (Viz struktura kódu.)

Jak napsat text?

Pokud chceme v pygame vykreslit text, musíme si nejdříve zvolit font. Můžeme si zvolit buď systémový font, nebo font vlastní.

Systémový font získáme pomocí příkazu font = pygame.font.SysFont(font_name, size). font_name je jméno fontu a size je velikost písma.

Vlastní font získáme pomocí příkazu font = pygame.font.Font(path, size). Kde path je cesta k .ttf souboru (souboru definujícímu font). Pokud path je None, tak se použije výchozí font.

Text získáme pomocí metody font.render. První parametr je řetězec určený k vykreslení, třetí je barva písma. Druhý parametr způsobuje zaoblení rohů písma, pokud je nastavený na True (tzv. antialiasing).

Metoda text.get_rect vrátí obdélník, ve kterém je text vypsán. Samotné vykreslení textu provedeme pomocí metody self.screen.blit, které předáme text a pozici pomocí objektu obdélníku.

Tato úloha již byla uzavřena a není možné ji odevzdat. Navazující úlohy ti však byly automaticky odemčeny. Navíc se můžeš podívat na vzorové řešení úlohy.
Piškvorky

Poté co Los Karlos konečně naprogramoval hrací desku, tak si uvědomil, že by s lihovkami stejně mohl hrát jen jednu hru. Protože Karlík má rád svůj počítač, tak mu nabídnul, že mu doprogramuje zobrazování hracích kamenů. Jenže Los Karlos je netrpělivý a rád by si co nejdřív zahrál, nebo aspoň rozbil Karlíkovi počítač. Tak Karlík má málo času a spěchá. Proto si zapomněl přečíst jak fungují události a teď si neví rady. Dokážete mu pomoct?

V následující úloze budete mít za úkol zpracovat události od uživatele. Budete mít kostru programu. Vaším úkolem je do programu doplnit do objektu Game metody check_events a draw_background.

Metoda draw_background vykreslí hrací pole o výšce a šířce self.size (self.size určuje počet políček v řádku a sloupci). Pro vykreslení políčka použijte pro určení jeho rozměrů konstantní hodnotu CELL_SIZE. Dejte si pozor, metoda by měla nechat okraj na každé straně okraj délky CELL_SIZE. Pokud si nejste jistí, jestli je hrací, plocha vykreslena správně, tak se podívejte do předchozí úlohy. Jestliže si ani pak nebudete vědět rady, napište do diskuzního fóra. (Ujistěte se ale, že nikdo před vámi nedostal odpověď na totéž. Na druhou stranu pokud vám moje odpověď na otázku nepomůže, zkuste otázku položit jinak a dejte mi co nejpřesněji vědět, co vám není jasné, ať to můžu vysvětlit.) Metoda nic nevrací ani nevolá funkci pygame.display.update(), ta se volá později po vykreslení hracích kamenů. Metoda je podobná předchozí úloze ovšem tady je hodnota screen (možná i některé další) uložena v objektu. (Získáte ji pomocí self.screen namísto screen.)

Další metodu, kterou máte za úkol implementovat je check_events. Tato metoda má za úkol zpracovat tyto události od uživatele:

  • pygame.QUIT, pokud nastane tato událost, pak metoda vrátí string "quit".

  • pygame.MOUSEBUTTONUP, zavolejte funkci self.play a předejte ji pozici políčka na které myš klikla. Pozice je tuple ve tvaru (x,y)(x,y). Indexujeme od nuly. Nemusíte řešit, zda je pole plné, nebo zda se kliklo mimo hranice desky. Funkce v takovém připadě nic neprovede. Tato událost neovlivňuje návratovou hodnotu metody.

  • pokud nenastala událost pygame.QUIT, pak metoda vrátí string "self".

Tato úloha nebude opravována automaticky. Před odevzdáním se ujistěte, že se hra běží, jak má. Body dostanete až na konci série (společně s velkými úlohy). Kostru souboru si můžete stáhnout zde. Odevzdáváte jeden soubor s příponou .py.

Řešení

Metody draw_stones můžou vypadat napřiklad takto:

    def draw_stones(self):
        for i in range(1, self.size + 1):
            for j in range(1, self.size + 1):
                if self.array[i - 1][j - 1] == 'X':
                    pygame.draw.ellipse(self.screen, X_COLOR,
                                        pygame.Rect(CELL_SIZE * i,
                                                    CELL_SIZE * j, CELL_SIZE,
                                                    CELL_SIZE))
                if self.array[i - 1][j - 1] == 'O':
                    pygame.draw.ellipse(self.screen, O_COLOR,
                                        pygame.Rect(CELL_SIZE * i,
                                                    CELL_SIZE * j, CELL_SIZE,
                                                    CELL_SIZE))

    def check_events(self):
        result = "self"
        for event in pygame.event.get():
            if event.type == pygame.QUIT:
                result = "quit"
                return result
            if event.type == pygame.MOUSEBUTTONUP:
                position = list(pygame.mouse.get_pos())
                position[0] //= CELL_SIZE
                position[0] -= 1
                position[1] //= CELL_SIZE
                position[1] -= 1
                position = tuple(position)
                self.play(position)
        return result

První metoda je podobná funkci z minulé úlohy. V metodě check_events jsme pro událost MOUSEBUTTONUP získali pozici myši. Následně jsme vydělili velikostí buňky, abychom získali pozici políčka. Nakonec jsme provedli korekci, kvůli okrajům.

Pokud se chcete podívat jak kód vypadá v celku, chcete si zahrát piškvorky, nebo chcete si zkusit naprogramovat strategii, níže vám zveřejňuji zdrojový kód.

import pygame

CELL_SIZE = 30
BRIGHT_COLOR = (230, 230, 230)
DARK_COLOR = (30, 30, 30)
BLACK = (0, 0, 0)
X_COLOR = (255, 0, 0)
O_COLOR = (0, 0, 255)

class Game:
    def __init__(self, size, screen):
        self.array = []
        for i in range(size):
            self.array.append([' '] * size)
        self.actual_symbol = 'X'
        self.screen = screen
        self.size = size

    def in_grid(self, coords):
        if not (coords[0] < len(self.array) and coords[0] >= 0 and
                coords[1] < len(self.array) and coords[1] >= 0):

            return False
        else:
            return self.array[coords[0]][coords[1]] == ' '

    def add_symbol(self, symbol, coords):
        if self.in_grid(coords):
            self.array[coords[0]][coords[1]] = symbol
        else:
            raise ValueError

    def __repr__(self):
        return str(self.array)

    def __str__(self):
        return str(self.array)

    def _check_five_at_pos(self, i, j):
        symbol = self.array[i][j]
        all_5 = True
        for di in range(5):
            if i + di < len(self.array):
                all_5 = all_5 and self.array[i + di][j] == symbol
            else:
                all_5 = False
        if all_5:
            return symbol
        all_5 = True
        for dj in range(5):
            if j + dj < len(self.array[i]):
                all_5 = all_5 and self.array[i][j + dj] == symbol
            else:
                all_5 = False
        if all_5:
            return symbol
        all_5 = True
        for dij in range(5):
            if i + dij < len(self.array) and j + dij < len(self.array[i]):
                all_5 = all_5 and self.array[i + dij][j + dij] == symbol
            else:
                all_5 = False
        if all_5:
            return symbol
        all_5 = True
        for dij in range(5):
            if i + dij < len(self.array) and j - dij >= 0:
                all_5 = all_5 and self.array[i + dij][j - dij] == symbol
            else:
                all_5 = False
        if all_5:
            return symbol
        return None

    def play(self, position):
        if not self.in_grid(position):
            return False
        if self.check_five() != ' ':
            return False
        self.add_symbol(self.actual_symbol, position)
        if self.actual_symbol == 'X':
            self.actual_symbol = 'O'
        else:
            self.actual_symbol = 'X'
        return True

    def check_five(self):
        for i in range(len(self.array)):
            for j in range(len(self.array[i])):
                symb = self._check_five_at_pos(i, j)
                if symb == 'X' or symb == 'O':
                    return symb
        return ' '

    def check_events(self):
        result = "self"
        for event in pygame.event.get():
            if event.type == pygame.QUIT:
                result = "quit"
                return result
            if event.type == pygame.KEYDOWN:
                if self.check_five() != ' ':
                    result = "menu"
            if event.type == pygame.MOUSEBUTTONUP:
                position = list(pygame.mouse.get_pos())
                position[0] //= CELL_SIZE
                position[0] -= 1
                position[1] //= CELL_SIZE
                position[1] -= 1
                position = tuple(position)
                self.play(position)
        return result

    def draw_text(self, text, x, y):
        font = pygame.font.Font(None, 25)
        text = font.render(text, True, BRIGHT_COLOR)
        text_rect = text.get_rect(center=(x, y))
        self.screen.blit(text, text_rect)

    def draw_background(self):
        for i in range(1, self.size + 1):
            for j in range(1, self.size + 1):
                if i % 2 == j % 2:
                    color = BRIGHT_COLOR
                else:
                    color = DARK_COLOR

                pygame.draw.rect(self.screen, color,
                                 pygame.Rect(CELL_SIZE * i, CELL_SIZE * j,
                                             CELL_SIZE, CELL_SIZE))

    def draw_stones(self):
        if self.check_five() != ' ':
            self.draw_text(self.check_five() +
                           " won. Press any key to continue.",
                           (self.size + 2) * CELL_SIZE // 2, CELL_SIZE // 2)

        for i in range(1, self.size + 1):
            for j in range(1, self.size + 1):
                if self.array[i - 1][j - 1] == 'X':
                    pygame.draw.ellipse(self.screen, X_COLOR,
                                        pygame.Rect(CELL_SIZE * i,
                                                    CELL_SIZE * j, CELL_SIZE,
                                                    CELL_SIZE))
                if self.array[i - 1][j - 1] == 'O':
                    pygame.draw.ellipse(self.screen, O_COLOR,
                                        pygame.Rect(CELL_SIZE * i,
                                                    CELL_SIZE * j, CELL_SIZE,
                                                    CELL_SIZE))

    def draw(self):
        self.draw_background()
        self.draw_stones()

    def game_tick(self):
        result = self.check_events()
        self.draw()
        pygame.display.update()
        return result


def game(size=15):
    pygame.init()
    screen = pygame.display.set_mode((CELL_SIZE*(size + 2),
                                      CELL_SIZE*(size + 2)))

    done = False
    game_screen = Menu(screen)
    while not done:
        result = game_screen.game_tick()
        if result == "self":
            continue
        if result == "quit":
            done = True
        if result == "menu":
            game_screen = Menu(screen)
        if result == "game":
            game_screen = Game(size, screen)

    pygame.quit()


if __name__ == "__main__":
    game(15)
