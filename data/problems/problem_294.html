Zadání

Brno renovuje linky MHD a zadalo tuto práci Karlíkovi.

Karlík vymyslel několik klíčových otázek, které mu pomůžou v informovaném a daty podloženém rozhodování. Jednotlivé podúkoly dobře definoval, ale nyní by se potřeboval věnovat obchodní stránce věci. Rozhodl se [:autsôrsovat:] řešení na tebe. Původně chtěl Karlík platit v Charlescoinech, zrovna se ale dost propadly na burze a tak budeš odměněn v měně mnohem stabilnější: Ξpointech.

Zadání úlohy

Formálně: Brno je definováno jako Seznam ulic (hran ve spojitém neorientovaném grafu) reprezentovaných slovníky s hodnotami pod klíči a (ID první křižovatky), b (ID druhé křižovatky), w (délka ulice). Brno je konečně velké. Brno nemusí být planární graf (obsahuje tunely, mosty i raketu).

Edges = List[Dict[str, int]]

Ukázka

edges = [
    {'a': 1, 'b': 2, 'w': 4},
    {'a': 0, 'b': 2, 'w': 42},
    {'a': 1, 'b': 0, 'w': 16},
]

Reprezentuje město o 3 křižovatkách s ID ∈{0,1,2}∈{0,1,2} spojené ulicemi délek 4, 16 a 42 (trojúhelníková nerovnost není splněna protože ulice nemusejí být rovné).

Použij šablonu

Aby bylo tvé řešení kompatibilní s Karlíkovým systémem, musíš použít tuto šablonu a to bez jejího přejmenování. V případě nekompatibility tě Karlík neodmění žádnými Ξpointy.

V kostře jsou i pomocné funkce na vytvoření a vizualizaci grafu pracující s korektním grafovým formátem.

1. podúloha

DPMB má k dispozici speciální kyvadlové linky, které můžou střídavě jezdit z jednoho konce ulice na konec druhý (podobně jako v úloze Kreditová jízdenka). Tyto linky může dopravce nasadit na libovolné ulice. Cílem tohoto systému by bylo obsloužit každou křižovatku. DPMB však nemá neomezené finance, proto chce vybudovat minimální délku těchto kyvadlových linek.

Formálně: Vytvoř funkci task1(edges: Edges) -> Edges, která v grafu ulic města (edges) najde souvislý podgraf, obsahující všechny vrcholy s minimálním celkovým součtem délek hran. Navraťte podgraf ve formátu Edges.

Řešení

from heapq import heapify, heappush, heappop
from typing import List, Tuple, Union, Dict

from graphviz import Digraph, Graph

Edges = List[Dict[str, int]]


def unique_vertices(edges: Edges):
    return max({edge['b'] for edge in edges}) + 1


def dict2tuple(edge):
    return edge['w'], edge['a'], edge['b']


def tuple2dict(edge):
    return {'a': edge[1], 'b': edge[2], 'w': edge[0]}


def graph_cost(edges: Edges):
    return sum([edge['w'] for edge in edges])


def graph_eccentricity(edges: Edges, center: int):
    """Graph has to be a tree"""

    visited = [{'': float('inf')} for _ in range(unique_vertices(edges))]


def visualize(all_edges: Edges, selected_edges: Edges, main=None):
    dot = Graph()
    for i in range(unique_vertices(all_edges)):
        if i == main:
            dot.node(str(i), style='filled', fillcolor='grey')
        else:
            dot.node(str(i))

    for edge in {dict2tuple(e) for e in all_edges} - {dict2tuple(e) for e in selected_edges}:
        e = tuple2dict(edge)
        dot.edge(str(e['a']), str(e['b']), label=str(e['w']), color='grey')

    for edge in selected_edges:
        dot.edge(str(edge['a']), str(edge['b']), label=str(edge['w']), color='red')

    return dot


def task1(edges: Edges):
    """Basically spanning tree"""

    v = unique_vertices(edges)
    visited = [False for _ in range(v)]
    adj_list = [[] for _ in range(v)]

    for edge in edges:  # fill adjacency list for each vertex
        e = dict2tuple(edge)
        adj_list[edge['a']].append(e)
        adj_list[edge['b']].append(e)

    heap = adj_list[0]  # start at node 0
    heapify(heap)  # init heap

    selected_edges = []

    while len(heap) > 0:
        smallest = heappop(heap)  # remove and work with the currently smallest edge
        w, a, b = smallest[0], smallest[1], smallest[2]
        new = None
        if not visited[a]:
            new = a
        if not visited[b]:
            new = b
        if new:
            visited[new] = True
            selected_edges.append(tuple2dict(smallest))
            for neighbor in adj_list[new]:
                if not visited[neighbor[1]] or not visited[neighbor[2]]:
                    heappush(heap, neighbor)

    return selected_edges


def task2_comp(edges: Edges, center: int):
    """Basically Dijkstra"""

    v = unique_vertices(edges)
    # Each vertex stores the best edge that leads to it
    visited = [(float('inf'), -1, -1, -1) for _ in range(v)]
    visited[center] = (0, center, center, 0)
    adj_list = [[] for _ in range(v)]

    for edge in edges:  # fill adjacency list for each vertex
        # extra information so we can return Edges with weights from the original graph
        e = *dict2tuple(edge), edge['w']
        adj_list[edge['a']].append(e)
        adj_list[edge['b']].append(e)

    heap = adj_list[center]  # start at center
    heapify(heap)

    while len(heap) > 0:
        smallest = heappop(heap)
        w, a, b, w_old = smallest
        new = None
        if visited[a] > smallest:
            new = a
        if visited[b] > smallest:
            new = b
        if new is not None:
            visited[new] = smallest
            for neighbor in adj_list[new]:
                neighbor = neighbor[0] + w, *neighbor[1:4]
                if visited[neighbor[1]] > neighbor or visited[neighbor[2]] > neighbor:
                    heappush(heap, neighbor)
    return visited[:center] + visited[center + 1:]


def task2_post(visited: List[Tuple[float, int, int, int]]):
    return [tuple2dict((w_old, a, b)) for w, a, b, w_old in visited]


def task2(edges: Edges, center: int):
    return task2_post(task2_comp(edges, center))


def task3(edges: Edges):
    min_eccentricity, centroid, subgraph = float('inf'), -1, None
    v = unique_vertices(edges)

    for center in range(v):
        visited = task2_comp(edges, center)
        eccentricity = max([w for w, a, b, old_w in visited])
        if min_eccentricity > eccentricity:
            min_eccentricity = eccentricity
            centroid = center
            subgraph = visited

    return task2_post(subgraph), centroid

Bonusová úloha

Tato bonusová úloha byla zadaná Los Karlosem. Věta > Zadání stejné jako 2. podúloha, ale lehčí vypočítat, protože místo neomezeně velkého počtu autobusů můžeš použít autobusů pouze 10.

je silně zavádějící, protože omezením počtu autobusů je úloha převeditelná na problém obchodního cestujícího, který je NP-těžký a velmi podobná vehicle routing problem.

Náznak redukce: Mějme graf GG pro který chceme najít minimální cestu (pro lehčí redukci, viz hemiltonovská cesta × hemiltonovská kružnice). Přidejme do GG nový vrchol Z ze kterého povede do každého z ostatních vrcholů hrana o váze rovné součtu všech vah v G. Vytvořme graf G’, který bude 10 grafů G a nový vrchol S. Vrchol S bude spojen s Z_i vahou o velikosti součtu všech vah v G. Spousťme task4 na grafu G’ s počátkem v S.

Z toho důvodu bylo jakékoliv funkční řešení uznáno, byť i pro graf o 10 vrcholech výpočet trvá 26 minut.


def eval_one_bus(adj_list: AdjList, vertices: Tuple[int]):
    total_time = 0
    for v_a, v_b in zip(vertices, vertices[1:]):
        if adj_list[v_a][v_b] is None:
            return None
        total_time += adj_list[v_a][v_b][0]

    return total_time


def eval_plan(adj_list: AdjList, perm: Tuple[int], where_to_divide: Tuple[int], center: int):
    total_time = 0
    for start, end in zip(where_to_divide, where_to_divide[1:]):
        # every bus must start in the center
        vertices = center, *perm[start:end]
        res = eval_one_bus(adj_list, vertices)
        if res is None:
            return None
        total_time = max(res, total_time)

    return total_time


def create_graph(adj_list: AdjList, perm: Tuple[int], where_to_divide: Tuple[int], center: int):
    edges = []
    for start, end in zip(where_to_divide, where_to_divide[1:]):
        # every bus must start in the center
        vertices = center, *perm[start:end]
        for v_a, v_b in zip(vertices, vertices[1:]):
            edges.append(tuple2dict(adj_list[v_a][v_b]))

    return edges


def task4(edges: Edges, center: int) -> Edges:
    """Basically travelling salesman problem, thus NP-complete"""

    v = unique_vertices(edges)
    # adj_list = [[] for _ in range(v)]
    adj_list = [[None for _ in range(v)] for _ in range(v)]

    for edge in edges:  # fill adjacency list for each vertex
        e = dict2tuple(edge)
        adj_list[edge['a']][edge['b']] = e
        adj_list[edge['b']][edge['a']] = e

    # all vertices without center
    no_center = list(range(v))
    no_center.remove(center)

    best_time = None
    best_solution = None
    for perm in permutations(no_center):
        for bus_count in range(9):
            for vertice_intervals in combinations(range(v), bus_count):
                vertice_intervals = 0, *vertice_intervals, v
                res = eval_plan(adj_list, perm, vertice_intervals, center)
                if res is None:
                    continue
                if best_time is None or res < best_time:
                    best_time = res
                    best_solution = perm, vertice_intervals
    if best_solution is None:
        return []
    return create_graph(adj_list, *best_solution, center)
