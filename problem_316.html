Zadání

Doposud jsi se naučil dělat programy, které umí dělat spoustu užitečných věcí, ale mají jeden velký problém. Veškerá data, se kterými pracují, jsou uložená pouze v RAM a po skončení programu zmizí. Ty bys ale chtěl, aby ti data vydržela déle než po dobou běhu programu a jistě přemýšlíš, jak na to. Odpovědí na tuto otázku jsou soubory.

Práce s textovým souborem

Než začneme s nějakým souborem pracovat, potřebujeme ho nejdříve otevřít. k tomu nám poslouží with as block, jehož syntaxi můžete vidět zde:

with open("file.txt") as file:
    # Zde můžeš se svým souborem pracovat
# Zde již bude soubor uzavřen a nebode možné s ním dále pracovat

Hlavní výhodou with as bloku je, že po jeho skončení se soubor vždy uzavře a to i v případě, že uvnitř dojde k výjimce, tedy není potřeba řešit uzavírání souboru pomocí try/except bloků. Kód výše ti otevře soubor s názvem "file.txt" a vytvoří objekt file, pomocí kterého budete moct souborem pracovat. Pomocí kódu výše otevřeš soubor v read-only módu, tedy do něho nebudeš moci zapisovat. Pokud chceš do souboru něco napsat, musíš zadat metodě open jako druhý parametr mód, pomocí kterého soubor otevřeš. Tudíž pokud chceš vytvořit soubor a něco do něj napsat, otevři soubor následujícím způsobem:

with open("file.txt", "w") as file:
    # Tady můžeš něco do souboru napsat
# Zde již bude soubor uzavřen a nebode možné s ním dále pracovat
PŘEHLED VŠECH MOŽNÝCH MÓDŮ PRO OTEVŘENÍ
MÓD ANGLICKY FUNKCE
r read čtení (nelze zapisovat), výchozí hodnoty atributu mode
w write zápis (přepíše původní obsah souboru)
a append zápis na konec (původní obsah zůstává, nový se připojí nakonec souboru)
r+ read čtení a zápis
w+ write zápis a čtení (přepíše původní obsah souboru)
a+ append zápis na konec a čtení
x exclusive zápis (jako w), ale selže pokud soubor existuje


Nyní máme soubor otevřený a můžeme s ním začít pracovat. K tomu nám poslouží objekt file, který obsahuje několik metod, pomocí kterých můžeme se souborem pracovat:

  • text = file.read([n:int]): Načte n znaků ze souboru a uloží je do proměnné text. Pokud máme k dispozici méně než n znaků, pak načte tolik znaků, kolik máme k dispozici. Pokud n neuvedeš, načte ti veškerý text v souboru.
  • text = file.readline(): Načte ti 1 řádek souboru a nahraje ho do proměnné text.
  • text = file.readlines(): Načte ti celý soubor do proměnné text ti nahraje List stringů, kde každý prvek seznamu představuje 1 řádek souboru.
  • file.write(text:str): Napíše obsah proměnné text do souboru
  • file.writelines(text:List[str]): Projde všechny prvky ze seznamu předaného jako parametr a každý z nich napíše do souboru na vlastní řádek.

Objekt File obsahuje samozřejmě daleko více metod a atributů, se kterými můžete pracovat, a uvedené funkce také můžou obsahovat více nepovinných argumentů. Na to, co všechno tato třída nabízí, se můžeš podívat například zde

Všechny metody, které čtou celé řádky je čtou včetně koncového symbolu \n, případně \r\n
Pokud se pokusíš otevřít pro čtení soubor, který neexistuje, dostaneš výjimku.

Zde můžete vidět nějaké příklady práce se soubory:

# Vytiskne všechny řádky souboru file.txt, které nejsou prázdné, do konzole
with open("file.txt") as file:
    for raw in file.readlines():
        if raw.replace("\r", "").replace("\n", "") != "":
            print(raw, end="")

# Vytiskne prvních 10 řádků souboru
with open("file.txt") as file:
    for i in range(10):
        print(file.readline())

# Zkopíruje obsah souboru source.txt do souboru destination.txt
with open("source.txt") as fileFrom:
    with open("destination.txt", "w") as fileTo:
        fileTo.write(fileFrom.read())

# Spadne s chybou, jelikož jsi otevřel soubor pro čtení, ale pokoušíš se do něj zapsat
with open("file.txt") as file:
    file.write("Hello world")

Práce se složkami

Pro práci se složkami je potřeba importovat balíček os, tento balíček nám nabízí mimo jiné soubor metod, pomocí kterých můžeme pracovat se složkami:

  • os.listdir([directory:str]): vrátí seznam všech souborů a složek ve složce directory. Pokud neuvedeme žádný parametr, vypíše obsah aktuální složky.
  • os.mkdir(directory:str): vytvoří složku directory
  • os.rmdir(directory:str): smaže složku directory

Samozřejmě metod pro práci se složkami a soubory je podstatně více, na celkový jejich seznam se můžeš podívat třeba zde.

Pár příkladů kódu pro práci se složkami:

import os

# vytvoří složku A
os.mkdir("A")
# vytvoří složku b ve složce A
os.mkdir("A/B")
# smaže složku B ze složky A
os.rmdir("A/B")
# vytvořenní složky s absolutní cestou
os.mkdir("/home/karlik/Documents/A") # Linux/macos
os.mkdir("C:/Users/karlik/Documents/A") # Windows
# vypsání obsahu složky A
os.listdir("A")
# vypsání obsahu složky s absolutní cestou (Linux)
os.listdir("/home/karlik/Documents")

Standardy pro ukládání textových dat

Nyní již umíme vytvářet adresářové struktury a textové soubory, tak se pojďme podívat na to, jak do těchto souborů uložit nějaká zajímavá data. Pro ukládání dat v textové formě existuje spousta různých konvencí, které mají své výhody i nevýhody, z nichž nejrozšířenější jsou CSV, XML a JSON. 2 z těchto 3 konvencí si nyní stručně popíšeme.

CSV

Zkratka CSV pochází z anglického comma separated values, což je asi nejvýstižnější popis celé konvence. Data, uložená v csv souboru představují tabulku, kde 1 řádek souboru představuje 1 řádek tabulky a na oddělení dat v jednotlivých sloupcích si zvolíš nějaký znak jako oddělovač (obvykle ,, ;, &, |, $ apod.) a pomocí tohoto znaku rozdělíš data do jednotlivých sloupečků. Tedy následující tabulka představující např. databázi uživatelů nějaké vaší aplikace by v csv souboru vypadala třeba takto (jako oddělovač jsem použil ;):

ID NAME SURNAME
0 KARLÍK TUČŇATÝ
1 LOS KARLOS
2 SOB KARSOB 
0;KARLÍK;TUČŇATÝ
1;LOS;KARLOS
2;SOB;KARSOB

Jedná se ovšem o hrubé zjednodušení csv konvence, přesnou definici si můžeš přečíst zde.

V Pythonu by se ti na práci s csv soubory mohly hodit následující 2 funkce:

  • text = str.split(sep): tato funkce ti rozseká string str podle oddělovače sep a vrátí ti pole s hodnotami seřazenými stejně jako ve stringu str
  • str = sep.join(text): Tato funkce očekává jako parametr text list stringů, které spojí do 1 jediného dlouhého stringu pomocí oddělovač sep.

JSON

CSV je sice krásné svojí jednoduchostí, ale pro uložení komplexnějších dat není příliš praktické. Pro tyto účely však máme k dispozici XML či JSON. Pokud umíte programovat v jazyce JavaScript, můžete si zjednodušeně JSON představit jako JavaScriptový objekt uložený v textovém souboru. Samozřejmě pokud JavaScript neznáš, tak ti vysvětlíme, jak přesně to funguje.

JSON obsahuje 2 hlavní elementy, řekněme objekt a pole. Pole začíná a končí hranatými závorkami [] a obsahuje několik různých dat, která můžou být jak stringy, čísla, logické hodnoty…, tak JSONové objekty nebo jiná pole, které jsou odděleny čárkami. Objekt začíná a končí složenými závorkami {}, a obsahuje atributy, které jsou odděleny také čárkami. Atributy jsou ve formátu název:hodnota, kde název je libovolný řetězec představující název atributu a hodnota může být, řetězec, číslo, datum, …, případně jiný JSON-ový objekt nebo nějaké JSON-ové pole. Každý atribut musí obsahovat jak název tak hodnotu a žádný objekt nesmí obsahovat 2 nebo více atributů se stejným názvem. Celý JSONový soubor poté obsahuje 1 kořenový objekt, případně 1 kořenové pole.

Na příkladu si ukážeme, jak můžeme v JSONu reprezentovat například strukturu IT oddělení.

{
    "name":"ITDepartment",
    "id":"LD0012",
    "room":"11.12.4",
    "number_of_employees":5,
    "manager":{
        "name":"Karlík",
        "surname":"Tučňatý",
        "id":12
    },
    "programmers":[
        {
            "name":"Los",
            "surname":"Karlos",
            "id":123
        },
        {
            "name":"Sob",
            "surname":"Karsob",
            "id":4
        }
    ],
    "sysAdmins":[
        {
            "name":"Los",
            "surname":"Rudolph",
            "id":64
        }
    ]
}

Pokud se ti nechce vytvářet a načítat JSON soubory ručně, pak existuje knihovna json pro práci s JSONovými daty. Tato knihovna ti umožní předělat JSONová data do Pythonu tak, že z JSONových objektů ti udělá slovník, kde názvy atributů budou jeho klíče s odpovídajícími hodnotami a z JSONového pole ti udělá klasický list.

Tato knihovna ti také nabízí vytvoření JSONového souboru ze struktury slovníků a polí a obráceně. Z této knihovny bys mohl využít tyto funkce:

  • str = json.dumps(obj): Z objektu obj, jehož formát je popsaný výše vytvoří string, představující JSONonovou reprezentaci obj.
  • obj = json.load(file): Načte obsah z JSONového souboru a převede jej na Pythonovou kompozici slovníků a polí
  • obj = json.loads(str): Stejný, jako json.load(...), akorát místo souboru s JSONovým obsahem očekává string, ve kterém je celý JSON načtený.

Příklad pro práci s JSONem v Pythonu s využitím výše uvedené reprezentace IT oddělení:

import json

# Metoda, která přidá nového systémového administrátora do IT oddělení
def addSysadmin(jsonFile, name, surname, id):
    newUser = {"name":name, "surname":surname, "id":id}
    with open(jsonFile) as file:
        itDep = json.load(file)
    itDep["sysAdmins"].append(newUser)
    with open(jsonFile, "w") as file:
        file.write(json.dumps(itDep))
Tato úloha již byla uzavřena a není možné ji odevzdat. Navazující úlohy ti však byly automaticky odemčeny. Navíc se můžeš podívat na vzorové řešení úlohy.
