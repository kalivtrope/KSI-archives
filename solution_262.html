Řešení

Nejdříve si ukažme, jak vypadá začátek veřejného vstupu, protože na něm budeme ukazovat jednotlivé techniky.

a
aa
aaa
aah
aahed
aahing

Zadání napovídalo, že můžeš začít prostým vypsáním trie, například v podobě výpisu stromu.

|- 
|-- a$
|--- a$
|---- a$
|---- h$
|----- e
|------ d$
|----- i
|------ n
|------- g$

Tento výpis obsahuje vše, co potřebujeme k znovuobnovení příslušné částí původního slovníku. Zanoření symbolizuje, že se jedná o potomka předchozího prvku. Symbol $ symbolizuje, že prvek je akceptující. Z této ukázky je tedy vidět, že obsahuje slova: a, aa, aaa, aah, aahed, aahing. Ač to na první pohled nevypadá jako anglická slova, skutečně jsou na začátku slovníku, na kterém testujeme.

Takový výpis je zhruba 3-4x větší než původní slovník, rozhodně se tedy nejedná o efektivní kompresi. Obsahuje ale spoustu nepodstatných informací. Víš, jak jsem před chvilkou zmínil, že zanoření o další úroveň reprezentuje, že prvek je dítětem? Co takhle předpokládat, že se se budu zanořovat vždy, pokud najdou písmeno a vynořovat právě když uvidím nějaký specifický symbol (třeba <). Tím místo výpisu několika - před každým slovem jenom občas budeme muset vypsat < když se nebudeme chtít vynořit, nebo několik < když se budeme chtít vynořit o víc úrovní.

a$
a$
a$
<h$
e
d$
<<
i
n
g$

Takový výpis je hned o dost kratší. Další věc, kterou můžeme bezpečně odstranit, je řádkování. To skutečně žádnou informaci nenese.

a$a$a$<h$ed$<<ing$

Další technikou, které můžeme využít, je fakt, že máme velmi omezený povolený vstup. Zadání garantuje, že to budou jenom malá písmena anglické abecedy, do kódovaného výstupu ale můžeš použít jakékoliv tisknutelné ASCII znaky. Co takhle si teda přidat informaci, zda je prvek akceptující, přímo do znaku samotného? Uděláme to velmi jednoduše - pokud je akceptující, uděláme z něj velké písmeno.

AAA<HeD<<inG

To se nám ten vstup ale krásně zmenšil! Byť jen pomocí těchto technik jsme schopni uložit vstup na méně než poloviční velikost a tedy získat až 7 bodů.

Ještě lepší komprimace, abys mohl dostat i více bodů, se dá udělat taky spoustou způsobů. Mohls například skládat více informací do jednoho znaku, jako jsme to udělali výše s velkými písmeny. Já jsem si třeba všiml, že ve svém komprimovaném výstupu jsou sekce jako tato:

uM<<<<<<<<dermatitiS<<<<<<<<<<iD<<

Na můj vkus je tam trochu moc ‘<’ a trochu málo písmen. Co s tím? Prostě si spočítám, kolik je tam < a nahradím je všechny odpovídajícím číslem!

uM8dermatitiS10iD2

A voila, už je můj výstup zhruba třetinový oproti původnímu slovníku. A už je dokonce těsně menší, než kdybych původní slovník jenom zazipoval na Linuxu. (Windows 10 má komprimační parametry nastavené trošku jinak a stále nás asi o 5% poráží.) Máme ještě spoustu prostoru, kam to zlepšovat, ale tohle už bohatě stačí na plný počet bodů.

Základní testovací framework (kód) 
from solutionNew_participant import compress_output, decompress_output # solutionNew_participant je nazev tveho reseni, tzn napr. soubor solutionNew_participant.py # nebo nasledujici vymen za nasledujici radek
# from trie_golf import compress_output, decompress_output

import datetime
import logging

import os

input_folder_path = "."
first_run = True

def makedir_if_not_exist(directory):
    if not os.path.exists(directory):
        os.makedirs(directory)

logging.basicConfig(level=logging.INFO, filename='hodnoceni.txt', filemode='w', format='%(asctime)s :: %(message)s')

def load_input(filename):
    with open(filename) as f:
        content = f.readlines()
    return content

def output_file(content, filename="default_output_name.out"):
    with open(filename,"w+") as f:
        f.write(content)
        #print(content)

def plaintext_to_wordlist(lines):
    output = set()
    for x in lines:
        current_word = x.strip().lower()
        if len(current_word) == 0:
            continue
        if current_word.isalpha() == False:
            continue
        output.add(current_word)
    return output

def sort_set_to_list(wordlist):
    return sorted(list(wordlist))

def set_to_output_file(wordlist, output_filename):
    a = sort_set_to_list(wordlist)
    output = ""
    for x in a:
        output += x + "\n"
    output_file(output, output_filename)

def sort_file(input_filename, output_filename):
    set_to_output_file (plaintext_to_wordlist(load_input(input_filename)), output_filename)


def run_tests(input_filename="words_alpha.in", bar=0.6):
    logging.info(input_filename)
    short_name_of_input = input_filename[:4]

    makedir_if_not_exist("output")
    print("[.]", input_filename)

    if first_run:
        sort_file(input_folder_path+input_filename, input_folder_path + "proccessed/" + short_name_of_input+"_00_input_sorted.out")

    content_input_array_of_lines = load_input(input_folder_path + "proccessed/"+ short_name_of_input+"_00_input_sorted.out")
    content_input_set = plaintext_to_wordlist(content_input_array_of_lines)

    startime = datetime.datetime.now()
    content_compressed_plaintext = compress_output(content_input_set)
    content_decompressed_set = decompress_output(content_compressed_plaintext)
    endtime = datetime.datetime.now()

    output_file(content_compressed_plaintext, "output/"+short_name_of_input+"_01_compressed_plaintext.out")
    set_to_output_file(content_decompressed_set, "output/"+short_name_of_input+"_02_decompressed_input.out")

    sum_len_of_words =  sum(len(x) for x in content_input_set)

    try:
        assert len(content_input_set.difference(content_decompressed_set)) == 0, "compression and decompression doesn't result in the original set - A"
        assert len(content_decompressed_set.difference(content_input_set)) == 0, "compression and decompression doesn't result in the original set - B"
    except Exception as e:
        logging.error(e)
        set_to_output_file(content_input_set.difference(content_decompressed_set), "output/"+short_name_of_input+"_03a_set_diff.out")
        set_to_output_file(content_decompressed_set.difference(content_input_set), "output/"+short_name_of_input+"_03b_set_diff.out")
        logging.warning("[x] Difference on input and output sets, diff writen to files.")

    compression_ratio = len(content_compressed_plaintext) / sum_len_of_words
    logging.info("Achieved compress ratio: {}".format(str(compression_ratio)))

    assert len(content_compressed_plaintext) < sum_len_of_words, "compressed content plaintext is bigger than noncompressed"
    logging.info("Took time: {}s".format(str((endtime - startime).total_seconds())))
    if (compression_ratio <= bar):
        logging.info("[.] +3b")

try:
    run_tests("shakespeare.in", 1)
    run_tests("words_alpha.in", 0.6)
    run_tests("xato-net-10-million-passwords-1000000.in", 0.5)
except Exception as e:
    logging.error(str(e))
except KeyboardInterrupt:
    logging.error("Keyboard interupt")
